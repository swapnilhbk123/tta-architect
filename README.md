# ✈️ TTA Group | AI Itinerary Architect

This is a custom internal tool designed for **TTA Group DMC** to automate the creation of professional travel itineraries and quotations. By combining the power of Google Gemini AI with high-quality PDF generation, it turns rough travel notes into client-ready proposals in seconds.

## 🌟 Key Features
- **AI Beautification:** Automatically transforms short, raw notes into elegant, descriptive travel scripts using Google Gemini.
- **TTA Branding:** Generates PDFs with the official Navy & Gold color scheme.
- **DMC Logistics:** Built-in fields for Pax count, Destination, Hotel details, and Pricing.
- **Cost Efficient:** Runs entirely on the Free Tier of GitHub, Streamlit, and Google AI.
